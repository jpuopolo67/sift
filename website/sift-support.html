Frequently Asked Questions

How do I install Sift?

Visit the Chrome Web Store, search for "Sift Bookmark Manager," and click "Add to Chrome." The extension will appear in your browser toolbar.

How does duplicate detection work?

Sift normalizes URLs before comparing them. This means it treats the following as duplicates:

  • http:// and https:// versions of the same URL
  • URLs with and without www.
  • URLs with trailing slashes
  • URLs with tracking parameters (utm_source, fbclid, etc.)

This catches duplicates that look different but point to the same page.

What is the Health Score?

The Health Score (1-100) reflects the overall state of your bookmark collection. Points are deducted for:

  • Duplicate bookmarks
  • Dead links (broken URLs)
  • Stale bookmarks (not visited recently)
  • Uncategorized bookmarks in root folders
  • Poor folder organization

A higher score means cleaner, better-organized bookmarks.

How does dead link checking work?

When you click "Check Dead Links," Sift tests each bookmarked URL in the background. It sends HTTP HEAD requests to verify the links are still working. Results are cached to avoid checking the same URLs repeatedly.

Note: Some websites may block automated requests, which could result in false positives. Review the results before deleting.

What are "stale" bookmarks?

Stale bookmarks are pages you haven't visited in a long time (default: 180 days). You can adjust this threshold in the extension settings. Stale detection uses your browser history to determine when you last visited each bookmarked page.

How do AI features work?

The AI categorization feature uses Claude (by Anthropic) to analyze your bookmarks and suggest a logical folder structure. To use it:

  1. Get a Claude API key from console.anthropic.com
  2. Enter your API key in Sift's settings page
  3. Click "Restructure Folders & Bookmarks" in the Actions tab

Sift creates a new organized folder structure inside a "Sift" folder - your original bookmarks are never modified or deleted.

Is my data private?

Yes. All bookmark analysis happens locally in your browser. We don't collect, store, or transmit your bookmark data. The only external communication is:

  • Dead link checking: HTTP requests to your bookmarked URLs to verify they're working
  • AI features (optional): Bookmark titles and URLs sent to the Claude API using your own API key

See our Privacy Policy for complete details.

Can I undo changes?

Sift does not have a built-in undo feature. However:

  • Duplicate removal: Keeps the most recently added bookmark
  • AI categorization: Creates copies in a new folder - originals are untouched
  • Deletions: Cannot be undone - review carefully before confirming

We recommend exporting your bookmarks (Chrome Menu > Bookmarks > Bookmark Manager > Export) before making large changes.

Troubleshooting

The extension isn't showing any bookmarks

Try these steps:

  1. Click the refresh button (â†») in the extension header
  2. Close and reopen the extension popup
  3. Disable and re-enable the extension in chrome://extensions
  4. Restart Chrome

Dead link check is slow

Dead link checking requires testing each URL, which takes time. The check runs in the background so you can close the popup and continue browsing. You'll receive a notification when it completes.

Results are cached, so subsequent checks will skip recently-tested URLs.

AI categorization failed

Common causes:

  • Invalid API key: Verify your Claude API key in settings
  • Insufficient credits: Check your Anthropic account balance
  • Rate limiting: Wait a few minutes and try again
  • Network issues: Check your internet connection
